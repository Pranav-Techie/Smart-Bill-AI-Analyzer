# 💡 Smart Bill AI Analyzer

A smart web application that helps users **analyze, track, and manage bills efficiently** using AI-like logic and intelligent insights.

---

## 🚀 Features

### 🧾 Bill Analysis

* Extracts:

  * Bill Type (Electricity, Water, Internet, etc.)
  * Amount
  * Due Date
* Works with **text input and voice input**

---

### ⚠️ Smart Reminder System

* Detects upcoming due dates
* Alerts:

  * Due TODAY
  * Due within next 3 days

---

### 📊 Spending Insights

* Calculates **total spending**
* Detects **increase in spending compared to previous bills**
* Helps users identify **overspending patterns**

---

### 🎙 Voice Input

* Speak bill details instead of typing
* Uses browser Speech Recognition

---

### 📜 Bill History

* Stores all bills locally
* View previous entries anytime
* Option to clear history

---

### 🖼 Image Upload (UI Ready)

* Upload bill images (future AI vision support)

---

## 🧠 What Makes This Different?

Unlike traditional bill apps that only store data, this project focuses on:

* 🧠 **AI-like understanding of raw text**
* ⚡ **Instant insights without manual entry**
* 📊 **Spending pattern detection**
* 🔔 **Proactive reminders instead of passive tracking**

👉 It acts like a **lightweight personal financial assistant**, not just a storage tool.
